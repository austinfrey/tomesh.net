# tomesh.net
Website for Toronto Mesh.

## Usage

### 1. Install dependencies

Install the Jekyll gem:

```bash
$ gem install jekyll
```
**Windows users:** [Run Jekyll on Windows](http://jekyll-windows.juthilo.com/)

### 2. Running locally

```bash
$ jekyll serve
```
